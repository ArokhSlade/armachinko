# Armachinko
![Armachinko](/image/armachinko_header_cropped_2.png Armachinko)
My first semester project at S4G.   
As a Team of 7 we built a quirky pinball spin-off, where players take direct control of the ball \- in the shape of a shotgun-wielding armadillo chasing the highest ~~score~~ bounty.
